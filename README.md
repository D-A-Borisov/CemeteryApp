# Настройка проекта

## 1. Получите API ключ Яндекс Карт
- Перейдите на [Yandex Cloud Console](https://console.cloud.yandex.ru/)
- Создайте сервисный аккаунт
- Активируйте Yandex Maps API
- Получите API ключ

## 2. Настройте local.properties
Создайте файл `local.properties` в корне проекта: YANDEX_MAPS_API_KEY=ваш_ключ_здесь

## 3. Синхронизируйте проект
